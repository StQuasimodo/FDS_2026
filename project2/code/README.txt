The program is devided into 4 source code files: A+B.c, tree.c, collect.c, output.c.
The main function is in A+B.c. Functions related to binary search trees are included in tree.c. 
Functions used to collect data and remove repeated elements are included in collect.c. 
Functions to find a and b and print preorder binary search tree are included in output.c. 
Make sure  the four files are in the same folder when you run A+B.c.
